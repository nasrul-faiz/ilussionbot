📘 How Custom Commands Work
Trigger — command yang user taip di WhatsApp. Contoh: .hello

Response — mesej balasan bot (boleh teks, emoji, caption media).

Buttons — sokong semua type di README package: quick reply, URL, call, copy, dan single select (list).

Category — untuk susun command dalam dashboard (General, Fun, Utility, dll).

Works in — group dan private chat.

Priority — custom command jalan selepas built-in command, jadi tak override command asal bot.
💡 Tip: guna \n untuk line break. Contoh: Line 1\nLine 2
